## What you need to know

* You can set Upload folder for any file, want to upload file to server.
* You can set Download folder for download file from server.

## Configure FileCloud

![](/img/Kizspy_Screenshot_5.png)

| Setting | Description |
| --- | --- |
| <kbd>Upload Folder</kbd>| The folder Kizspy will upload file |
| <kbd>Download Folder</kbd> | The folder Kizspy will download file |
| <kbd>Auto Extract after Download</kbd> | Enable auto extract file after download |
